ccminer-tpfuemp — credits & donation
=====================================

Lineage
-------
Originally based on Christian Buchner's and Christian H.'s CUDA project
(inactive on GitHub since 2014), then:

    tpruvot/ccminer  ->  Kudaraidee/ccminer-kudaraidee  ->  tpfuemp/ccminer-tpfuemp (this fork)

This fork tracks the ccminer-kudaraidee 1.3.0 / tpruvot 2.3.1 base and targets
modern NVIDIA hardware on CUDA 11.8.

Kernel / code contributors
---------------------------
Includes kernels and work from: lenis0012, tpruvot, alexis78, djm34, djEzo,
tsiv, and krnlx, plus the many contributors to the upstream ccminer projects.
